## 📌 Sobre o Repositório

Este repositório foi criado para armazenar e organizar todos os códigos, listas de exercícios, trabalhos práticos e anotações desenvolvidos durante a disciplina de **Fundamentos de Algoritmos**, do curso de **Engenharia de Software** da **Universidade Estadual de Maringá (UEM)**.

O objetivo principal é documentar minha evolução na lógica de programação e na resolução de problemas algorítmicos utilizando **Python**.

---

## 📚 Tópicos Abordados

Durante a disciplina, exploramos os seguintes conceitos de lógica e sintaxe Python:

* **Introdução:** Variáveis, tipos de dados embutidos (`int`, `float`, `str`, `bool`) e operadores.
* **Estruturas Condicionais:** `if`, `elif` e `else`.
* **Estruturas de Repetição:** Laços `for` e `while`.
* **Estruturas de Dados Básicas:** * Listas (e listas de listas / matrizes)
  * Tuplas
  * Dicionários
* **Strings:** Manipulação, formatação (`f-strings`) e fatiamento (*slicing*).
* **Modularização:** Criação e chamada de funções (`def`), escopo de variáveis e retorno de dados.

## 🚀 Como Executar os Códigos

Para rodar os scripts deste repositório na sua máquina, você precisará ter o [Python](https://www.python.org/downloads/) instalado.

1. Clone o repositório:
   ```bash
   git clone [https://github.com/](https://github.com/)[SEU_USUARIO_AQUI]/[NOME_DO_REPOSITORIO].git
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd [NOME_DO_REPOSITORIO]
   ```

3. Execute o arquivo desejado usando o terminal:
   ```bash
   python nome_do_arquivo.py
   ```
   *(Nota: Se estiver usando macOS ou Linux, pode ser necessário usar `python3` no lugar de `python`)*
